Site Report

Introduction and Learning Experience

Creating this portfolio website has been a valuable learning experience that helped me understand the fundamentals of web development. At the beginning of the module, I had limited knowledge of HTML and CSS and found it difficult to understand how web pages were structured. Through regular practice and experimentation, I gradually became more confident in using semantic HTML elements and linking multiple pages together to form a complete website.

Challenges Faced During Development

One of the biggest challenges I faced was working with CSS. Small mistakes often caused layout issues, which required careful debugging and testing. This helped me improve my problem-solving skills and understand responsive design techniques.

Website Development Process

The website was developed as a multi-page portfolio including a home page, project page, contact page, video demonstration page, and this site report. Semantic HTML elements were used to improve structure and accessibility.

Design Choices and Validation

A clean and minimal design was chosen to ensure easy navigation and professional appearance. All HTML and CSS files were validated using online validation tools to ensure they meet web standards. This project strengthened my understanding of web development and provided a solid foundation for future learning.
